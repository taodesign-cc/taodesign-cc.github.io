# 作品图片放这里

把你的作品图片（jpg / png / webp 等）放进这个文件夹 `public/works/`。

放好后，在作品数据里用以下路径引用（注意以 `/works/` 开头，不要写 `public`）：

- 例如文件 `public/works/coffee-01.jpg` → 在数据里写 `"/works/coffee-01.jpg"`
- 缩略图建议单独准备一张较小的，如 `"/works/coffee-cover.jpg"`

也可以直接用在线图片链接（https 开头）。

命名建议：用英文/数字，避免空格，例如 `project1-01.jpg`、`project1-02.jpg`。
